Menue principale

La barre de menue principale contient deux menues déroulants.

File

File contient tout ce qui est relatif à la génération du code, à la sauvegarde demodule, ainsi qu'au lancement de la simulation.

  • New : Permet d'ouvrir une nouvelle fenêtre avec une scène vide afin de pouvoir faire un nouveau module.
  • Generate : Permet de générer le code du module courrant. Cette étape est necessaire lors de la création d'un module ouvert.
  • Compile : Permet de créer les fichier .class
  • Execute : Permet de lancer la simulation. Execute effectue aussi un Generate et un Compile.
  • Close : Permet de fermer la fenêtre courrante.

Edit

Edit permet d'éditer les propriétés du cadre générale du module. En cliquant sur "Edit Couple" il est possible de choisir le nom du futur module, ainsi que le nombre d'entrées et de sorties qui le compose
